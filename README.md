This app use Node.js, Express and Mongoose. Application de gestion d'entrainement sportif.

# Commands 

### Version
1.1.0

### Installation

```sh
$ npm install
```

### Launch app

```sh
$ node app
```

### Test

```sh
$ npm run test
```
### Build assets

```sh
$ npm run build
```
